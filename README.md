## Circus Trein
Een algoritme opdracht waar het de bedoeling is om zo efficiënt mogelijk dieren in wagons te vervoeren. Dieren bestaan uit 3 typen:
- Carnivoren
- Herbivoren
- Omnivoren

En 3 verschillende maten:
- Klein
- Medium
- Groot

Carnivoren mochten niet bij dieren die kleiner zijn dan hen, anders zouden ze worden opgegeten.

## IndividueelProject
In semester 2 was de opdracht om een web applicatie te bouwen met CRUD operaties en een verbinding heeft met MSSQL. De applicatie GameAttic maakt het mogelijk voor gebruikers om games toe te voegen, op te halen, te bewerken en te verwijderen. Deze games kunnen toegevoegd worden aan een eigen collectie om bij te houden welke games een gebruiker in bezit heeft. 

Ideeën die ik in een later stadium had willen ontwikkelen:
1. Collectie beheer
2. Game rating/review
3. Favorite/unfavorite
