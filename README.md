# IndividueelProject
'Project GC' is een prototype naam voor de webapplicatie om een game collectie bij te houden.
Het doel is dat mensen een overzichtelijke weergave krijgen van games en de waarde van de game collectie kunnen bekijken. Gebruikers kunnen games toevoegen met specifieke details zoals een titel, platform, release date, genre en prijs. Om games toe te voegen aan een persoonlijke collectie dienen gebruikers zich te registreren en in te loggen. Gebruikers kunnen hun profiel en collectie pagina personaliseren naar eigen smaak.
